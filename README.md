# KrishiShiksha

KrishiShiksha is a frontend digital learning platform designed to provide
simplified agricultural education to farmers.

## Features
- Curated content on soil health, crop planning, and government schemes
- AI-assisted advisory simulation using Google Gemini API concepts
- Multilingual support (English / Hindi)
- Accessibility-first, responsive UI
